# AI-Powered Code Assistant

A production-ready coding assistant built with RAG (Retrieval-Augmented Generation) for accurate, context-aware suggestions.

## Architecture

- Vector database for code embedding storage
- Custom retrieval pipeline for finding relevant context
- LLM integration for natural language understanding
- VS Code extension for seamless integration

## Results

Reduced developer lookup time by 40% in internal testing.
