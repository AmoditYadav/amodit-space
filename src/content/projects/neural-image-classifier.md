# Neural Network Image Classifier

This project implements a state-of-the-art image classification system for medical imaging applications.

## Key Features

- Custom CNN architecture optimized for medical images
- Data augmentation pipeline for handling limited training data
- Model interpretability with Grad-CAM visualizations
- REST API for easy integration

## Technical Details

Built with PyTorch and trained on GPU clusters, achieving 95% accuracy on held-out test data.
